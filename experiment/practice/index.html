Let's Learn Extended Argument Syntax!
Call the function using the textbox given there and see the output and observations given below.
The function on the right calculates the sum of all the arguments passed to it. It does this using the *args approach to take in an arbitrary number of arguments.
Fill in the blank with a comma separated list of numbers to find out the use of variable arguments.
Run the program
def sum(*numbers): total = 0 for number in numbers: total += number return total print(sum())
Submit
Output is:
Observations
sum
1
